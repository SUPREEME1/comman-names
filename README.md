# comman-names

Comman name. 
The user opens the web application in their browser, enters a name into the input field, and clicks the “Check” button. The application displays a message indicating that it is checking the name’s popularity and fetches data from the Nationalize.io API. If the name is found, the application shows the name’s popularity status (very common, somewhat common, or quite unique), the most likely origin, and the popularity score. If the name is not found, the application informs the user that the name appears to be unique. The user can then enter another name to check or close the application.
